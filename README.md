# Установка

`composer install`

`cp .env.example .env`

`nano .env` - настроить подключение к базе данных

`php artisan key:generate`

`yarn`

`yarn run prod` - сборка проекта

`php artisan migrate:refresh --seed` - это тестовые данные, выполнять только для локального окружения
